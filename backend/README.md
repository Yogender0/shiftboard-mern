# Shift Board - Backend (Express + MongoDB)
## How to run
1. Install dependencies:
   ```bash
   cd backend
   npm install
   ```
2. Create `.env` from `.env.example` and set `MONGODB_URI` and `JWT_SECRET`.
3. Start server:
   ```bash
   npm run dev
   ```
